# BizForHer Website

A Vite + React + TypeScript web app for BizForHer.

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui (Radix UI)
- React Router

## Run locally

### Prerequisites

- Node.js 18+ (Node 20+ recommended)
- npm (comes with Node)

### Setup

```bash
# 1) Install dependencies
npm install

# 2) Start the dev server (runs on port 8080)
npm run dev
```

Open the app at:

- http://localhost:8080

## Build for production

```bash
# Create an optimized production build
npm run build

# Preview the production build locally
npm run preview
```

## Assets

Static images used by the site live in `public/uploads/` and are referenced via `/uploads/<filename>`.
